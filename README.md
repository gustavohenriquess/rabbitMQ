## Generic Commands

to start RabbitMQ Server:

```
rabbitmq:start
```

to stop RabbitMQ Server:

```
rabbitmq:stop
```

on root Folder:

```
yarn install
```

```
node sampleSrc/send.js
```

to receive a message:

```
node sampleSrc/receive.js
```
